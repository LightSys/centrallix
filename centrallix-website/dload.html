Download Centrallix...

To obtain the most recent release copy of Centrallix, visit our

SourceForge downloads page for Centrallix

We now are distributing the packages in "Everything" archives. Choose whether you want to build from source, from source RPM's, or whether you want to install a binary distribution, such as EL4 (RHEL4/CentOS4) or EL5 (RHEL5/CentOS5) or others that we may make available. Download the "Everything" tgz bundle that you need, and extract it to get the individual packages.

You MUST install at least the "centrallix", "centrallix-lib", and "centrallix-os" packages, otherwise things will not work! Build and install centrallix-lib and centrallix-os before building and installing the main centrallix package.

Be sure to read the release notes that comes with the download! Please note that "Development" (0.7.x, 0.9.x) releases have a potentially higher level of bugs and may have incomplete security frameworks.

Prebuilt RPM downloads have been tested on several RedHat/Fedora type Linux platforms. TGZ files do contain a "spec" file suitable for building RPMs.

Unless you're sure you know what you're doing, only "Stable" releases (1.0.x) should be used in production operations.

Additionally, we have started to use GitHub primarily instead of SourceForge

SHA1 Checksums and RPM Signatures...

On a Linux system, you can verify SHA1 checksums from the command line, using the sha1sum command. Just type:

sha1sum centrallix-*.rpm centrallix-*.tgz

RPM's for Centrallix 0.9.x and newer are signed with LightSys' RPM GPG signing key (0xCBD237C2) which you can download from us or get from a public keyserver such as pgp.mit.edu. Note: if you download the key from a keyserver, know that some versions of RPM have trouble with signed keys, and may import the key using an incorrect key ID.
