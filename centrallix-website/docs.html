Documentation Packages...

To learn more about Centrallix, you can download the centrallix-doc and centrallix-sysdoc packages. The centrallix-doc package contains information that is useful to someone wanting to develop an application on top of Centrallix. The centrallix-sysdoc package is internal system documentation, for developers of Centrallix itself (and other interested persons).
