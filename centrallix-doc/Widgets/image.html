image

type: widget/image

visual: yes

container: no

form element: yes

Overview:

The image widget displays a picture (image).

This widget can be a form element, in which case it will display an image as specified by the data from the form.

Usage:

The image widget can be placed inside any container that allows for visual widgets. Only connectors may be placed inside it.

Properties:

Property Type Description
aspect sting Determines wither the image can be stretched or must stay its original aspect ration.
fieldname string If using a form for the image path, the name of the column in the datasource you want to reference.
form string If using a form for the image path, this is the name of the form. If left unspecified, and fieldname is supplied, the default will be whatever form the image is inside.
height integer Height, in pixels, of the image.
text string The words that come up if the image object cannot find its source and display an image.
width integer Width, in pixels, of the image.
x integer X-coordinate of the upper left corner of the image, relative to its container.
y integer Y-coordinate of the upper left corner of the image, relative to its container.

Actions:

Action Description
LoadImage Displays the image to user.
Offset Reloads the image with new offset values.
Scale Reloads the image with a new scale value.

Events:

Event Description
Click This event occurs when the user clicks the checkbox. No parameters are available from this event.
MouseDown This event occurs when the user presses the mouse button on the checkbox. This differs from the 'Click' event in that the user must actually press and release the mouse button on the checkbox for a Click event to fire, whereas simply pressing the mouse button down will cause the MouseDown event to fire.
MouseMove This event occurs when the user moves the mouse pointer while it is over the checkbox. The event will repeatedly fire each time the pointer moves.
MouseOut This event occurs when the user moves the mouse pointer off of the checkbox.
MouseOver This event occurs when the user first moves the mouse pointer over the checkbox. It will not occur again until the user moves the mouse off of the checkbox and then back over it again.
MouseUp This event occurs when the user releases the mouse button on the checkbox.

Client Properties:

Property Type Description
scale integer Determines how much the image grows or shrinks compared to the original.
source string An OSML pathname for the location of the image (such as a png or jpg file).
xoffset integer Distance it is along the x axis from its parent widget.
yoffset integer Distance it is along the y axis from its parent widget.
