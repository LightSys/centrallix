chart

type: widget/chart

visual: yes

container: no

form element: no

Overview:

A chart widget is used to display charts and graphs of data. It uses the Chart.js library.

Usage:

The chart widget can be placed inside of any visual container, and will attach itself to any objectsource widget that contains it (whether directly or indirectly). Charts may not contain visual widgets.

Properties:

Property Type Description
chart_type string Currently supported types are "bar", "line", "scatter", and "pie". Defaults to "bar". Chart.js supports much more than this, so new types could be added with relative ease.
legend_position string The position of the legend on the chart. May be "top", "botton", "left", or "right".
objectsource string The name of the ObjectSource widget which will supply data for the chart. We recommend that you do not specify this directly but instead embed the chart widget within a parent ObjectSource, directly or indirectly. If not specified, the chart will look for an ObjectSource in its parents.
start_at_zero boolean Set to false if you want the y axis to start at the lowest data value. Default is true.
title string The title of the chart. Default is none.
titlecolor string A color, RGB or named, of the chart title.
title_size integer The size, in points, of the title font.

Child Properties:

(of widget/chart-series child widgets)

Property Type Description
chart_type string Chart.js has limited support for having multiple chart types on the same axes. The outer chart type must still be specified if this is set.
color string A color, RGB or named, for the chart element (ex: line) which represents this series.
fill boolean Set to false to remove the color fill beneath a line chart. Default is true.
label string A label for the data series.
x_column string Which column in the data should be used for the x axis? By default, the chart will pick one for you.
y_column string Which column in the data should be used for the y axis? By default, the chart will pick one for you.

(of widget/chart-axis child widgets)

Property Type Description
axis string Which axis is this? Possible values are "x" and "y". Default is "x".
label string A label for the axis. Default is none.

Sample Code:


		

$Version=2$
// Here is a simple chart.
chart "widget/chart"
	{
	x=0; y=0;
	width=483;
	height=650;
	chart_type="bar";
	title="Sample Chart";
	titlecolor="black";
	legend_position="right";

	series1 "widget/chart-series" { color="blue"; }

	series2 "widget/chart-series"
		{
		fill=no;
		chart_type="line";
		x_column="name";
		y_column="permissions";
		}

	x_axis "widget/chart-axis" { axis="x"; label="X"; }
	}
